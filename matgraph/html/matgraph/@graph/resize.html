resize

PURPOSE

resize(g,n) --- change the number of vertices in g to n

SYNOPSIS

function resize(g, n)

DESCRIPTION 

 resize(g,n) --- change the number of vertices in g to n
 note: if n is less than nv(g), vertices will be lost
 if the graph has an embedding, all new vertices are sited at the origin.
