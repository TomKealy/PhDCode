SYNOPSIS

function match_join(g,h1,h2)

DESCRIPTION 

 match_join(g,h1,h2) --- overwrite g with a graph formed from disjoint
 graphs g1,g2 with a matching added between them. The graphs h1 and h2
 must have the same number of vertices.
