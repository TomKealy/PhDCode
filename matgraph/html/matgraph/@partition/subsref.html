subsref

PURPOSE

subsref --- implements the notation p(v) and p(v,w)

SYNOPSIS

function vec = subsref(p,S)

DESCRIPTION 

 subsref --- implements the notation p(v) and p(v,w)
 p(v) returns the part containing v
 p(v,w) returns 1 if v and w are in the same part and 0 otherwise
